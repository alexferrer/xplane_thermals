Easy installation instructions: 

1: Install the python script plugin according to :
2: Click on the green [Clone or Download]  button on the top right of this page
3. Click on the blue [Download Zip] button
4. Unzip all the actual files (Not the directory they are on) onto your Xplane python plugin directory: 
   The directory should be something like: ... X-Plane 11/Resources/plugins/PythonScripts/

Your directory should end up looking like:
